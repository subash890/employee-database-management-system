# Employee Database Management System using Python, Flask & MySQL

## 📌 Project Overview
This is a simple web-based Employee Database Management System built using **Python**, **Flask**, **MySQL**, **HTML**, and **CSS**.

The project allows users to manage employee records through a web interface instead of a terminal-based application.

---

## 🚀 Features
- Add Employee
- View All Employees
- Update Employee Details
- Delete Employee
- Search Employee by Name or Department
- Store employee records in MySQL database

---

## 🛠️ Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS
